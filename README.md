# Разработка приложения печати графиков с использованием MVC и IoC
## Цель работы
Изучение принципов построения слабосвязанных приложений на основе внедрения зависимостей (Dependency Injection) и IoC-контейнера, а также освоение архитектурного шаблона MVC при разработке графического приложения для построения и печати графиков по данным из базы данных SQLite.

В ходе работы необходимо:

1) реализовать загрузку данных из SQLite;
2) построить графики по загруженным данным;
3) обеспечить возможность выбора различных типов графиков;
4) реализовать переключение визуального оформления графиков;
5) реализовать экспорт графиков в PDF;
6) использовать архитектуру MVC;
7) использовать внедрение зависимостей через IoC-контейнер.
## Функциональные возможности программы
Разработанное приложение предоставляет пользователю следующие возможности:
### Работа с файлами
просмотр файловой системы компьютера;
выбор каталога через дерево папок;
отображение содержимого выбранной папки;
выбор SQLite-файла для анализа.
### Работа с базой данных
автоматическое подключение к SQLite-базе;
определение таблицы данных;
считывание данных вида:
преобразование данных в структуру GraphPoint.
### Построение графиков
отображение линейного графика;
отображение столбчатой диаграммы;
автоматическое обновление графика после выбора файла.
### Изменение внешнего вида
цветной режим;
черно-белый режим.
### Печать
экспорт графика в PDF;
выбор пользователем пути сохранения файла.
## Сценарии поведения пользователя
### Сценарий №1. Построение графика
1) Пользователь запускает программу.
2) Выбирает папку в дереве каталогов.
3) В таблице отображаются файлы выбранной папки.
4) Пользователь выбирает SQLite-файл.
5) Программа загружает данные.
6) Строится график.
### Сценарий №2. Изменение типа графика
1) Пользователь выбирает тип графика:
2) BarChart;
3) LineChart.
4) Программа перестраивает график.
### Сценарий №3. Изменение стиля
1) Пользователь устанавливает флажок "Черно-белый график".
2) График перестраивается с новой темой оформления.
### Сценарий №4. Экспорт в PDF
1) Пользователь нажимает кнопку "Печать графика".
2) Выбирает путь сохранения.
3) Программа сохраняет график в PDF.
